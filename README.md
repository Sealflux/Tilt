## Description
A motion-controlled keychain game console. Built around the Seeed XIAO ESP32C3 with an SSD1306 OLED and MPU6050 accelerometer. Planning to make a ball tilt game probably

## Features

## Hardware
- Seeed Xiao ESP32-C3
- SSD1306 128x64 OLED (I2C)
- Tactile Push Button
- 3D accelerometer and 3D magnetometer
- Buzzer
- Mystery LiPo Battery(Haven't decided which one on amazon yet)
- 3D-printed case
